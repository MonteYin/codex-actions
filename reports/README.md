# Research Tracker Reports

This directory contains automated research opportunity reports generated by the Research Tracker workflow.

## Structure

```
reports/
├── {owner}/
│   └── {repo}.md      # Per-repo tracking report
├── failures.md        # Log of failed analysis attempts
└── README.md          # This file
```

## Report Format

Each repo report contains:

1. **Active Research Backlog** - Aggregated action items with status
2. **Daily Entries** - Analysis results by date
3. **Archived Entries** - Older entries (collapsed)

## Opportunity IDs

Format: `R-YYYY-MM-DD-NN`

- `R` = Research opportunity
- `YYYY-MM-DD` = Date identified
- `NN` = Sequential number for that day

## Status Values

- **Todo** - Not yet investigated
- **Researching** - Currently being explored
- **Done** - Investigation complete
